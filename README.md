# GameDev.tv Progress Color

A Chrome extension that lets you customize the progress bar color on GameDev.tv courses.

## Features

- Color picker popup to choose any color for the video progress bar
- Color persists across sessions
- Updates in real-time as you pick

## Install

1. Clone this repo
2. Open `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked** and select the repo folder

## Usage

Click the extension icon to open the color picker and choose your color.
